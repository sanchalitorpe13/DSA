# Data Structures and Algorithms in C++

This repository contains my solutions to various Data Structures and Algorithms (DSA) problems solved in C++.

The goal of this repository is to improve problem-solving skills, strengthen algorithmic thinking, and prepare for coding interviews and competitive programming contests.

- Linked Lists

- Binary Search Trees

- Recursion & Backtracking
- Searching Algorithms
- Sorting Algorithms
- Greedy Algorithms
- Dynamic Programming
- Graphs
- Tries
- Bit Manipulation
- Sliding Window
- Two Pointers
- Prefix Sum
- Segment Trees
- Disjoint Set Union (DSU)
- Competitive Programming Problems
